# Gerador de Horário Escolar

# Descrição
Projeto em Python que gera automaticamente um horário escolar com base nas disciplinas e nas matrículas dos alunos.

# Objetivo
Evitar conflitos de horário entre disciplinas utilizando uma abordagem heurística.

# Funcionalidades
- Leitura de arquivos de disciplinas e matrículas
- Identificação de conflitos entre disciplinas
- Geração de horários sem conflitos
